LWC (Lightning Web Component):

Create a component that uses a wire adapter to fetch the aggregate results using a SOQL query.

Pass that JSON data into the JavaScript logic shown above.

SOQL Queries needed:

SELECT dbt__Project__r.Name, COUNT(Id) FROM [Object] GROUP BY dbt__Project__r.Name

SELECT dbt__Activity__c, COUNT(Id) FROM [Object] GROUP BY dbt__Activity__c

SELECT dbt__Description__c FROM [Object] WHERE dbt__Date__c = THIS_YEAR (for the keyword analysis).
